What I've Do When I Got a Macbook

Posted by HanTing on Tuesday, January 21, 2020

離職後自行購入一台Mac筆電，趁著這個機會紀錄電腦的環境佈置。剛拿到Mac筆電的注意事項及環境安裝

購買Mac前

關於Macbook Pro保固的部分一定要了解，mac在購買60天以內都可以申請延長保固，保固時間從1年延長到3年，需要 9590 元方可延長保固。mac損壞時，維修的錢往往都要超過2000元。延長mac 保固 是一種選擇。

買電腦前一定要先了解，自己買電腦要做什麼用途？mac對於設計人、開發者是極其友善的開發環境，但對於想要玩遊戲、文書處理的人來說，windows便是比較好的選擇。

環境

當我拿到一台新的mac時，我安裝的環境為 chrome, postman, rbenv VSCode, iTerm, Typora, iPic。VSCode具有相當多套件可以安裝，以我電腦環境來說，一共裝了Sublime Text Keymap and Settings Importer, Rubocop, Gitlens, ERB, ERB Helper Tags。在iTerm上安裝了zsh和oh-my-zsh，而在Brew上安裝了git, mysql@5.7, redis

必安裝：Brew、Git、rbenv、rbenv 設為預設、mysql

使用VSCode編輯所需的套件

  • 使用 code .指令：連結1、連結2

  • GitLens：可以看某行程式碼是誰編輯過的

  • Rubocop：檢查 Rails 的 Coding Style，相關設定在這裏

  • ERB：讓 erb 變彩色

  • ERB Helper Tags：erb 快捷鍵

  • Sublime Text Keymap and Settings Importer：Sublime 快捷鍵

安裝 iTerm 編輯流程

  • 安裝流程請看 這裡、這裡

  • 如何停止每次開終端機都要打 source ~/.bash_profile

    open ~/.zshrc 打開.zshrc 並將 source ~/.bash_profile 加至最後一行即可

    下面的 code block 為.bash_profile 存放指令，主要為rbenv和code .

    # export PATH="/usr/local/opt/ncurses/bin:$PATH"

# Add Visual Studio Code (code)
export PATH="$PATH:/Applications/Visual Studio Code.app/Contents/Resources/app/bin"

export PATH="$HOME/.rbenv/bin:$PATH"
eval "$(rbenv init -)"

部落格

在2020年1月歷經一連串的面試後，開始想要建立自己的部落格，將文章以主題式的方式呈現出來，現成的部落格具有標籤、文章分類的功能，非常方便。

不過比較不方便的地方是，一直以來使用markdown語言的我來說，blogger不支援markdown對我來說真的不太方便，所以以下介紹在blogger如何使用markdown。目前找到最好的方式為，將markdown轉為html語言後，再貼進去blogger的原始碼。

  • 安裝Typora：本地端使用markdown
  • markdown 轉 html網站：連結
  • blogger 使用 markdown 說明
  • 好看的 blogger 範例：PJCHENder

後來發現 blogger 除了內部可選用的主題，還有其他主題可選。使用方式很簡單，進入編輯Html，將原本的xml刪除， 並將載下來的xml檔貼上即可。下圖為替換後的新主題

除了blogger以外，還有medium、wix、webnode 等平台，而除了現有平台外，Hugo, Hexo 能夠將markdown轉成html。
